# Ark Station 2.0 (Lab 3)

Симулятор управления космической станцией в 2942 году. Проект реализован в рамках Лабораторной работы №3 по курсу объектно-ориентированного программирования.

## О проекте
Это продвинутая версия симулятора, в которой весь код переработан с использованием **наследования** и **полиморфизма**. 

### Основные фичи:
- **Абстрактная архитектура**: Модули и Роботы реализованы через иерархию классов.
- **Динамика**: Объекты создаются и удаляются в реальном времени (`new`/`delete`).
- **Строительство**: Возможность возводить новые отсеки станции.
- **Модернизация**: Система апгрейдов уровней модулей (до 10 уровня).
- **Вариант 8**: Уникальный тип робота «Локальный вычислитель», не нагружающий общую сеть станции.

## Архитектура
Проект разбит на логические блоки:
- `Module`: Базовый класс для отсеков (Генератор, Архив и т.д.).
- `Robot`: Базовый класс для жителей (Интегратор, Хранитель и т.д.).
- `Station`: Ядро симуляции, управляющее списками указателей на объекты.

## Сборка и запуск
Требуется `g++` (C++17) и `make`.

```bash
make        # Сборка
./ark_station # Запуск
make clean  # Очистка
```
