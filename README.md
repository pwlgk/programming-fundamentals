# Лабораторные работы

Этот репозиторий содержит лабораторные работы по C++. Все программы компилируются с использованием `make`.

## Требования
Для сборки вам понадобится:
- Компилятор `g++` с поддержкой C++17
- Утилита `make`

## Лабораторная работа 1: Вычисление пройденного пути поезда метро (lab1)
Простая программа для расчета пути, пройденного поездом метро при равномерном движении и торможении.

### Как запустить
1. Перейдите в директорию `lab1`:
   ```bash
   cd lab1
   ```
2. Скомпилируйте программу с помощью `make`:
   ```bash
   make
   ```
3. Запустите исполняемый файл:
   ```bash
   ./metro_calc
   ```
4. Для очистки файлов сборки используйте команду:
   ```bash
   make clean
   ```
