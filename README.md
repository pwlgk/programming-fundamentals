# C++ Programming Labs

Репозиторий содержит цикл лабораторных работ по курсу объектно-ориентированного программирования.

## Структура проекта
- **lab1**: Основы C++, простая калькуляция.
- **lab2**: Введение в классы и объекты.
- **lab3 (Ark Station 2.0)**: Симулятор космической станции. Наследование и полиморфизм.
  - *Особенности*: Вариант 8 (Локальный вычислитель).
- **lab4 (Ark Station 2.1)**: Продвинутая версия симулятора с перегрузкой операторов.
  - *Особенности*: Вариант 2 (Детектор клонов), Синтез сознания, Модульная сборка.

## Технологии
- Язык: C++17
- Сборка: Makefile / g++
- ОС: Linux / POSIX

## Запуск (на примере Lab 4)
```bash
cd lab4
make
./ark_station
```
